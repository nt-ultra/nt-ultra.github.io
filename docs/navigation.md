# Navigation

New Tab Ultra uses some standard methods for navigating around the page. This document will attempt to list all of the "not so obvious" methods.

### Shortcuts

- Right Clicking a shortcut -> shows an extended list of options
- Clicking a shortcut -> opens the link in the current tab
- Middle clicking a shortcut -> opens the link in a new tab
- Dragging shortcuts around -> rearranges them in the shortcut grid
- Dragging a shortcut into a folder -> adds it to the folder

### Folders

- Right clicking a folder -> shows an extended list of options
- Clicking a folder -> open its contents
- Middle clicking a folder -> opens all links inside, in new tabs
- Dragging folders around -> rearranges them in the shortcut grid
- Dragging a folder shortcut around -> rearranges them in the folder
- Dragging a folder shortcut out of the folder -> removes it from the folder

### Trackers

- Right clicking a tracker -> shows an extended list of options
- Clicking a tracker -> opens its link in a new tab

### Tab Browser

- Right clicking a tab -> shows an extended list of common tab options
- Right clicking a Section Header "`Window 1 (10 tabs) ↓`" -> shows options for all tabs in that section
- Clicking a tab -> sends you to the tab
- Clicking a Section Header "`Tab Group 1 (10 tabs) ↓`" -> will expand/collapse the tabs in that Section
- Middle clicking a tab -> closes the tab
- Dragging a tab from one Window/Group to another Window/Group -> moves it there
- Dragging a tab from the Tab Browser to the Shortcut Grid -> turns it into a shortcut

### Searchbar

The search bar supports typical up/down, enter/esc, keyboard navigation
